Event 2:CODE ARENA

Date: Friday, September 29th

The Fundamentals of C Programming Workshop, part of Code Arena (INFINITY 2k24), was held on 29th September via Google Meet, organized by the ACM Student Chapter, SVPCET, and conducted by the Code Arena Team.

Coordinated by Paritosh Magare and Pratiksha Ghate, and led by Ayush Kshirsagar and Pratiksha Ghate, the workshop covered C programming basics, debugging, setting up coding environments, and solving real-world problems. With active participation and positive feedback, the workshop successfully prepared students for the challenges of Code Arena.
